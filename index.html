RubyStation

Manages your browser-GUI Ruby apps

What's RubyStation?

Ruby has many awesome Web frameworks - Rails, Sinatra, Merb, Ramaze, etc. Maybe you have written some Web applications with them. Then, how about desktop applications?

You can create a desktop application with Rails by:

  1. Write an app with Rails.
  2. ./script/server will start server on localhost, i.e. it does work without the Internet.
  3. Pack the files into a gem file.
  4. Users install the app with RubyGems, and run it on their localhost. Why can't we call it a desktop application? :-)

RubyStation manages the port numbers for each application, so all you need is click a link to start the application, and new browser tab is opened.

Dependencies

  • ruby >= 1.8.x (it's 1.9 ready!)
  • Mac or Unix (Windows is not supported yet)

Install

  1. Be sure you have rubygems >= 1.3.5 (gem --version) 
    $ gem --version
1.3.5
  2. Install gemcutter 
    $ sudo gem install gemcutter ; gem tumble
  3. Install RubyStaiton 
    $ gem install ruby-station
  4. Execute RubyStaiton 
    $ ruby-station
    Then config file
  5. Install an application (eg. LifeLog - online notepad)

    Enter "lifelog" to the input box and click [install]. Then 'gem install' is run, and new app is installed after some minutes.

    List of applications are available at wiki:Applications.

Write an application for RubyStaiton

Creating an app is just as easy as making a gem; See wiki:CreatingApps for more details.
